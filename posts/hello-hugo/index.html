使用Hugo生成静态博客

30 May 2025·2 mins

快速开始 #

前提条件 #

  1. 安装 Git
  2. 安装 Go (1.19或更高版本)

安装Hugo #

下载 Hugo ：hugo_0.145.0_windows-amd64.zip

解压，添加 hugo.exe 到系统环境变量

验证：

hugo version

初始化站点 #

选择一个目录作为hugo站点目录，例如在 D:\Hugo 目录下执行以下命令，站点目录为 D:\Hugo\Toaction

hugo new site Toaction

目录结构如下：

├── archetypes
│   └── default.md
├── hugo.toml         # 博客站点的配置文件
├── content             # 博客文章所在目录
├── data                
├── layouts             # 网站布局
├── static              # 一些静态内容
└── themes              # 博客主题

设置主题 #

以 congo 主题为例，在站点目录下，使用Git 安装主题

git init
git submodule add -b stable https://github.com/jpanther/congo.git themes/congo

将 congo主题文件夹下的 config 文件夹 themes/congo/config 复制到站点目录下

删除 hugo.toml 文件，修改 config/_default/hugo.toml 文件

# 顶端添加
theme = "congo"

# baseURL = "https://your_domain.com/"
defaultContentLanguage = "en"

enableRobotsTXT = true
summaryLength = 0

快速预览：

hugo server

部署到 Github #

新建仓库 #

在 Github 上新建一个 Repository，命名格式为 用户名.github.io，例如 Toaction.github.io

修改 config/_default/hugo.toml 文件中的 baseURL

theme = "congo"

baseURL = "https://Toaction.github.com/"
defaultContentLanguage = "en"

上传文件 #

使用命令 hugo 生成hugo静态页面，命令执行后，出现一个public文件夹

进入public文件夹，使用git上传文件

git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Toaction/Toaction.github.io.git
git push -u origin main

开始写作 #

创建模板 #

在 archetypes 目录下新建一个 posts.md，这是文章的 Front Matter 字段的模板，写入以下内容：

---
title: "{{ replace .Name "-" " " | title }}"
description: ""
date: {{ .Date }}
lastmod: {{ .Date }}
categories:
tags:
draft: false
---

新建文章 #

站点目录下终端输入

hugo new posts/hello-hugo/index.md

将会创建一个 hello-hugo 文章。使用 Hugo 的 Page Bundle 模式来管理静态资源，此时第一篇示例文章目录结构如下

.
├── archetypes
│   ├── default.md
│   └── posts.md
├── assets
├── config.yml
├── content
│   └── posts	# 文章目录
│       └── hello-hugo-start
│           ├── cover.webp	# 文章封面图片
│           ├── index.md	# 文章 Markdown 文件
│           └── hello.webp	# 其他文章图片
├── data
├── layouts
├── public
├── resources
├── static
└── themes
    └── congo

上传 #

使用 hugo 生成新的静态页面，并使用git push进行同步

git add .
git commit -m "add blog hello-hugo"
git push
